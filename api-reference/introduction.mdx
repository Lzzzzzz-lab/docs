## API格式约定

<Badge type="info" className="bg-blue-500 text-white px-2 py-1 rounded">
加密说明
</Badge>  
所有请求参数需组装成 JSON 格式请求体，使用 **RSA 公钥** 加密后，进行 **Base64 编码**。加密后的数据提交为 `data` 参数。

### 风险评分参考

<Badge type="warning" className="bg-yellow-500 text-white px-2 py-1 rounded">风险等级</Badge>

- 🟢 **低风险**：`0 ~ 30`，无风险或低风险。  
- 🟡 **中风险**：`31 ~ 64`，可能跟中高风险地址有过交易。  
- 🟠 **高风险**：`65 ~ 84`，建议隔离。  
- 🔴 **严重风险**：`85 ~ 100`，建议立即隔离。
