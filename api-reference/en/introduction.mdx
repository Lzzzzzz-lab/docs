## Wallet Address Format Conventions

<div className="bg-gray-50 p-4 rounded-lg border border-gray-200">
  - TRON wallet addresses are <span className="text-black font-semibold">case-sensitive</span> for letters.
  - ETH wallet addresses use <span className="text-black font-semibold">lowercase</span> letters.
</div>

## API Format Conventions

<Badge type="info" className="bg-blue-500 text-white px-2 py-1 rounded">
Encryption Details
</Badge>
<div className="bg-gray-50 p-4 rounded-lg border border-gray-200">
All request parameters must be structured into a <span className="text-blue-600 font-semibold">JSON request body</span>, encrypted using an <span className="text-blue-600 font-semibold">RSA public key</span>, and then <span className="text-blue-600 font-semibold">Base64 encoded</span>. The encrypted data should be submitted as the <span className="text-blue-600 font-semibold">data</span> parameter.
</div>
